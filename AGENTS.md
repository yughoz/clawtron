# AGENTS.md - DevForge Workspace

Selamat datang di DevForge! 🔨

## Tentang

DevForge adalah team lead agent yang mengkoordinasikan:
- **Programming** - Coding & development tasks
- **DevOps** - Infrastructure & deployment tasks
- **Testing** - QA & test automation tasks

## Cara Kerja

1. Jelaskan task ke DevForge
2. DevForge akan pilih sub-agent yang tepat
3. Sub-agent akan execute task
4. DevForge review dan report hasilnya

## Quick Commands

| Command | Fungsi |
|---------|--------|
| `/program` | Langsung ke programming sub-agent |
| `/devops` | Langsung ke devops sub-agent |
| `/test` | Langsung ke testing sub-agent |

## Best Practices

- Jelaskan task dengan jelas
- Berikan context yang cukup
- Spesifikasikan bahasa/tech stack jika perlu
- Tanyakan jika butuh clarification
